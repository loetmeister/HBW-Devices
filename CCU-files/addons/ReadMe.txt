Das AddOn hausbusde2.tar.gz beinhaltet alle aktuell verfügbaren Features für Homematic Geräte von Hasu-Bus.de.
Die CCU löscht bereits vorhandene Geräte bei einem Update, wenn neue Features hinzukommen oder sich ändern.
Die Geräte müssen dann nach dem Update neu angelernt und Programme evtl neu geschrieben werden.

Das AddOn hausbusde.tar.gz bleibt kompatibel mit der Version 2.00 und dort wird nur die FW Datei aktualisiert. 

Das AddOn hausbusde2Booter.tar.gz ist nur temporär einzuspielen um die Booter-FW auf den Geräten zu aktualisieren.
ACHTUNG: 
Nach einem Booter update hat das Modul keine Firmware mehr und bleibt ohne Funktion bis das AddOn hausbusde2.tar.gz wieder
eingespielt wird und ein Firmware update durchgeführt wird.
